# Sardia Frontend (V2)

React 19 + Vite 6 + Tailwind 4 frontend for the Sardia literary website.

## Run locally

**Prerequisites:** Node.js >= 18

1. Install dependencies: `npm install`
2. Copy `.env.example` to `.env.local` and set `VITE_API_BASE_URL` to your backend
3. Start the dev server: `npm run dev` (http://localhost:3000)
